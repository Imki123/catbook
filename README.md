# How to use

`npm run web`

# How to deploy to web

`npm run deploy`
% node version use 16 %

# How to Build to native app

Go to https://docs.expo.dev/build/setup/

### Ready to build

```
npm install -g eas-cli
eas login
eas build:configure
```

Change version, versionCode, buildNumber up in app.json

### Build to EAS

```
npm run build
OR
npm run build:ios
```

And deploy an app in APP STORE console.
